## About
Saya adalah seorang programmer yang telah berpengalaman pada rekayasa perangkat lunak
serta melakukan uji penerimaan menggunakan UAT untuk perangkat lunak yang saya buat. Pada web
development saya berpengalaman menggunakan framework PHP CodeIgniter dan Golang Echo serta
berpengalaman menggunakan database MySQL. Diluar web development saya juga berpengalaman
dengan konsep Object-Oriented Programming (OOP) menggunakan bahasa C# untuk membuat
aplikasi mobile menggunakan Engine Unity 3D.
